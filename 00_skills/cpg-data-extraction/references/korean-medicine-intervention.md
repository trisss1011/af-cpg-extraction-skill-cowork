# 한의 중재 추출 상세 기준 — 한약 (v2.0)

한의표준임상진료지침 개발 매뉴얼 Ver2.0 (2022) 기반.

> **범위**: 본 스킬은 한약 중재 논문만 다룬다. 침·뜸·복합중재는 대상이 아니며, 필요 시 별도 스킬에서 처리한다.

---

## 1. 한약 치료 추출 필드

한약 치료는 **`한의중재_한약` 시트에 원문 그대로 추출**한다 (v1.10~). 근거표 작성은 후속 단계(별도 스킬)에서 수행한다.

### 1A. `한의중재_한약` 시트 열 구성 (메타분석용 엑셀, v1.13~)

1논문 = 1행 (wide form). 주처방만 기록, 부처방은 note에 서술.

| 열 | 필드명 | 설명 | 예시 |
|----|--------|------|------|
| A | 번호 | 기본정보 시트와 동일 | 242 |
| B | study_id | 기본정보 시트와 동일 | Su_2016 |
| C | formula_name_kr | 처방명 (한글) | 양혈복맥탕 |
| D | formula_name_cn | 처방명 (한자) | 养血复脉汤 |
| E | formulation | 제형 | 탕약 / 과립 / 캡슐 / 환제 / 주사제 / NR |
| F | composition | 약재+용량 원문 (쉼표 구분) | `黄芪 25g, 香附 10g, 当归 15g, ...` |
| G | administration | 투여경로·용량·횟수 (v1.13 신설) | `경구, 탕약 수전 400mL, 200mL×2회/일` |
| H | note | 비고 | 부처방: 생맥산 / NR 사유 등 |

### 1B. F열(composition) 기재 규칙

1. **형식**: `약재명` + 공백 + `용량원문단위` (예: `麻黃 9g`, `마황 9g`)
2. **구분자**: 쉼표(`,`) + 공백으로 통일. 원문이 `、` `·` `；`이어도 쉼표로 변환.
3. **번역·단위환산 금지** — 약재명·용량 모두 **원문 언어·단위 그대로**. 한자 논문이면 한자, 한글 논문이면 한글.
4. **법제 표기 유지** — `生·炒·炙·蜜炙·制·焦·煨` 등 포제 접두어가 원문에 있으면 **그대로 기재**. 임의 제거·정규화 금지.
   - 원문 `生黄芪25g` → F열 `生黄芪 25g` (유지)
   - 원문 `炙甘草8g` → F열 `炙甘草 8g` (유지)
   - 원문 `炒枣仁15g` → F열 `炒枣仁 15g` (유지)
   - 후속 표준화·통합 분석 단계에서 별도 정규화 처리.
5. **공통 용량 표기는 풀어서** 개별 기록. 예: `人參、麥門冬、五味子 各9g` → `人參 9g, 麥門冬 9g, 五味子 9g` (H열에 `원문 '各9g' 풀어 기재` 명시).
6. **약재 전체 용량 미기재** (중성약·시판 처방에서 약재명만 나열된 경우): F열에 약재명만 쉼표 구분으로 나열 (예: `麻黃, 桂枝, 杏仁, 甘草`). H열에 `용량 미기재` 명시.
7. **처방 전체 구성 비공개** (성분 비공개 캡슐 등): F열 `NR`, H열에 사유 기재.

### 1C. 다중 처방 처리

- 기본: 1논문에 여러 처방이 쓰였으면 **주처방만 행 1개**. 부처방은 H열(note)에 텍스트로 서술.
- 예: `부처방: 생맥산(心氣虛 환자 대상, 心俞 수혈에 가미), 발작 시 速效救心丸 병용`
- **전원 동시투여 (v2.7)**: 모든 환자에게 2개(이상) 처방이 **동시 병용**되어 주/부 구분 근거가 없으면 → 둘을 **합쳐 하나의 주처방으로 간주**해 1행 기재. 각 처방의 조성·용량 모두 F열에 보존(처방 경계는 H열 note에 표기). 예: No.77 桂枝甘草汤 가감 + 稳心颗粒.
- **개별화 처방 (변증 분기) (v2.7)**: 메인 처방 없이 변증(또는 환자 특성)에 따라 처방을 각기 달리 적용한 경우 → 행을 나누지 않고 C/D열 처방명에 `개별화 처방 (변증 분기)`로 1행 통합 기재(개별 처방별 메타분석 불가, 통합 메타분석만). 변증별 처방 구성·용량은 H열 note에 서술. 예: No.66 변증 sub-group별 4개 처방 분기.

### 1D. 대조군(C) 한약 처리

- `한의중재_한약` 시트는 **치료군(E) 한약 중재만** 대상.
- KM vs KM 비교 연구에서 대조군이 한약인 경우는 현재 버전에서 제외 (별도 확장 시 고려).

### 1E. 처방 유형 식별

- **표준처방 / 가감처방 / 자체처방(경험방) 구분은 본 시트에 기록하지 않는다.**
- 구분이 필요하면 후속 분석 단계에서 별도 판정.
- 처방명(C·D열)과 composition(F열)만으로 후속 단계에서 기본방 대조 가능.

### 1F. 제형(E열) 구분

| 코드 | 원문 예시 |
|------|---------|
| 탕약 | 汤剂 / decoction / 전탕 |
| 과립 | 颗粒 / granule / 엑스제 |
| 캡슐 | 胶囊 / capsule |
| 환제 | 丸 / pill / 환 |
| 주사제 | 注射液 / injection |
| 기타 | 膏 / 散 / 酒剂 등 — 원문 그대로 기재 |

> **주의**: extract(과립·캡슐)와 decoction(탕약)은 반드시 구분한다. 약효 차이 및 메타분석 이질성 평가에 직결.
