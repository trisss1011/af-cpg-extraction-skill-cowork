## v2.2 (2026-04-17) — 아웃컴 W·X열 삭제, V열 이분형 공란 규칙 명시

- **W(data_class), X(effect_measure) 열 삭제** (아웃컴 시트 24열 → 22열)
  - W는 H~M(원시데이터) / N~Q(효과크기) 채움 여부로 100% 자동 추론 가능 → 후속 RevMan Export 스킬에서 산출
  - X는 effect_type(N열) → LOG_ 접두사 매핑 규칙이 기계적 → 마찬가지
  - 추출자 부담 경감, 중복 정보 제거
- **V(val2_type) 규칙 명시화**
  - 연속형 행: 기본값 `SD` (또는 `SE`) — 기존 유지
  - **이분형 행: 공란** (신설) — val2가 Total이므로 SD/SE 개념 부적용
  - 기존 "기본값 SD" 규칙으로 이분형 행에 SD가 관성적으로 채워지던 오추출 차단
- **SKILL.md 수정**
  - L282~284 아웃컴 열 구성 표: V 설명에 "연속형 전용" 명시, W·X 행 삭제
  - L330~346 "추가열 (V·W·X) 기재 규칙" → "V열 (val2_type) 기재 규칙"으로 축소 재작성
  - 향후 RevMan Export 스킬이 W·X 자동 산출한다는 안내 블록 추가
- **샘플 파일 버전업**: `sample_v2.1.xlsx` → `sample_v2.2.xlsx`
  - 아웃컴 시트 W, X열 제거 (22열)
